# 📰 Daily Radar

Daily Radar is a personalised news dashboard that creates a custom daily briefing using live RSS feeds based on a user's selected interests.

The goal of the project is to reduce information overload by presenting only the news that matters most to each individual in an accessible, responsive interface.

---

## Features

### 📰 Personalised Daily Digest
- Select multiple interests
- Choose sub-categories such as:
  - Football
  - Formula 1
  - Music genres
  - Book genres
- Generate a customised daily news briefing

---

### 🌐 Live RSS Feed Integration

Daily Radar gathers articles from multiple RSS feeds including topics such as:

- Technology
- AI
- Business
- Sport
- Science
- Entertainment
- Gaming
- Books

Articles are grouped into relevant categories based on their content.

---

### ⭐ Priority Topics

Users can pin favourite interests which become highlighted throughout the application.

Priority topics receive extra visibility within the generated digest.

---

### 📚 Read Later

Save interesting articles to a local Read Later collection.

Saved articles remain available between browser sessions using Local Storage.

---

### 📖 Digest History

Each generated digest is stored locally, allowing users to revisit previous daily briefings.

---

### ♿ Accessibility

Daily Radar includes several accessibility options:

- Dyslexia-friendly font
- High contrast mode
- Larger text
- Reduced motion
- Keyboard navigation
- Visible focus indicators

---

### 🎨 Themes

Users can personalise the appearance with:

- Light Theme
- Dark Theme
- System Theme

Theme preferences are automatically remembered.

---

### 📊 Dashboard & Insights

The dashboard includes:

- Articles analysed
- Articles selected
- Estimated reading time
- Reading goal progress
- Digest quality score
- Coverage statistics

---

### 💾 Local Storage

The application remembers:

- Selected interests
- Priority topics
- Accessibility settings
- Theme
- Read Later articles
- Digest history

No personal information is sent to a server.

---

### 📱 Responsive Design

Daily Radar is fully responsive and works across:

- Desktop
- Tablet
- Mobile devices

---
